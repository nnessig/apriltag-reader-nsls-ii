# Demo Resources

This folder contains the images used in the demo video for this project.

## Demo Video

A step-by-step walkthrough of the pipeline is available on the team OneDrive.

## How to Reproduce the Demo

1. Watch the demo video and follow along step-by-step
2. Use the images in this folder when the video shows adding the calibration images

## Contents

* `calibration_images/` – images used for camera calibration in the demo
* `tag_images/` – images used for AprilTag detection in the demo

## Notes

* These files are only used if you choose to follow the demo
* You can use your own images instead when running the pipeline
